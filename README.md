# Code Snippet Manager


Full-stack snippet manager (Node/Express + React/Vite + Tailwind) — lightweight and extendable.


## Setup


### Backend


1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env` and set `MONGO_URI`
4. `npm run dev`


### Frontend


1. `cd frontend`
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:5173`


## Notes
- The API base URL is set in `frontend/src/api.js`. Use `VITE_API_BASE` env var to change it for production builds.
- Add authentication, user accounts, and permissions if you need multi-user support.
